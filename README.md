# word list

分からなかった単語をその場で残して、あとで見返せる小さなWebアプリです。

**→ https://junkoma2.github.io/word-list/**

## なぜ作ったか

単語帳アプリはたくさんありますが、「記録する→復習する」のサイクルを意識させる作りのものが多く、普段使いには少し重く感じていました。
このアプリは「気になったときに素早く残す」ことだけを目的にしています。単語を追加して一覧で確認する。それだけです。

## 主な機能

- 単語・メモ・タグを入力してリストに追加
- 同じ単語を重複登録すると「何回気になったか」をカウント
- タグによる絞り込み・キーワード検索
- フラッシュカード形式のテストモード
- データのエクスポート・インポート（JSON）
- PWA対応（ホーム画面から起動できる）
- pull-to-refresh でアプリを更新

## 使い方

1. 分からなかった単語を「単語」欄に入力する
2. 意味や使い方などを「メモ」欄に入力する（任意）
3. 「追加」ボタンで保存する
4. タグをつけておくと、あとで絞り込める
5. 「テスト」ボタンで単語を1枚ずつ確認できる

## データとプライバシー

データはブラウザのlocalStorageにのみ保存されます。外部サーバーへの送信は一切ありません。
ブラウザの「サイトデータを消去」を行うとデータが消えます。定期的なエクスポートバックアップを推奨します。
