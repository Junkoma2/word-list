# Contributing

## 方針

- 入力の速さを優先する
- 覚える前に、まず残せることを大切にする
- 画面を複雑にしすぎない
- 変更は小さく、意図が分かる形で残す

## 公開リポジトリとしての注意

- API キー、トークン、パスワードをコミットしない
- 不要な依存関係を増やさない

## 動作確認

追加・検索/タグ絞り込み・絞り込み0件表示・インポート確認・テストモード進行の主要フローは、
`tests/` にjsdomベースの回帰確認を用意している（devDependencyのみで、公開する静的ファイルには影響しない）。

```bash
npm install
npm test
```
